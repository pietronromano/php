## VS Code: Run selected text in Active Terminal
To Configure:
- CTRL|CMD+SHIFT+P
    - Type"Preferences: Run Selected Text in Active Terminal"
    - Click on the Configuration icon
    
- Remove other associated Keys:
  - Press Keys:
  - On Windows: CTRL+E, 
  - On MAC: CMD+SHIFT+E
